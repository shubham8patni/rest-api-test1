Welcome to my First REST-API project !!


This API helps you to :
  • Get sum of list of numbers : {sum}
  • Get average of list of numbers : {avg}
  • Find if given value is a palindrome : {palin}

URL to call the api :
https://first-rest-api-sp.herokuapp.com/{function-name}/ls=[{list or values}]
Replace :
  • {function name} by sum or avg or palin based on requirement
  • {list or values} by list of number for avg and sum function and a single value for palin function

Examples :
  • https://first-rest-api-sp.herokuapp.com/sum/ls=[1,2,3]
  • https://first-rest-api-sp.herokuapp.com/avg/ls=[1,2,3,55,66,82,67]
  • https://first-rest-api-sp.herokuapp.com/palin/ls=[1234554321]


OUTPUT/Response: The response of the queries will be in form of JSON.
